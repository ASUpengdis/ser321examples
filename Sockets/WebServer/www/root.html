You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- returns the product of two numbers. Example request: /multiply?num1=1&num2=2
  • /github -- returns the owner's name, ID, and name of their public repositories. Example request: /github?query=users/amehlhase316/repos
  • /projectile -- returns the max height, max distance, and time of flight of a projectile. Example request: /projectile?angle=45&velocity=100
  • /dice -- returns a list of roll values, the larges roll and average roll value. Example request: /dice?dice1=6&dice2=20&rolls=10
